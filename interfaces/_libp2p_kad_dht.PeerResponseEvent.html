Interface PeerResponseEvent

Emitted when query responses are received form remote peers. Depending on the query these events may be followed by a FinalPeerEvent, a ValueEvent or a ProviderEvent.

interface PeerResponseEvent {

    providers: PeerInfo[];
    record?: DHTRecord;
