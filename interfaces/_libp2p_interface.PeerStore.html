    getInfo

    • getInfo(peerId: PeerId, options?: AbortOptions): Promise<PeerInfo>

      Returns a PeerInfo object for the passed peer id. This is similar to get except the returned value contains fewer fields and is often used to exchange peer information with other systems.

      The returned object can be passed to JSON.stringify without any additional processing.

      Parameters

      • peerId: PeerId
      • Optionaloptions: AbortOptions

      Returns Promise<PeerInfo>

      See

      https://docs.libp2p.io/concepts/fundamentals/peers/#peer-info

      Example

      const peerInfo = await peerStore.getInfo(peerId)

      console.info(JSON.stringify(peerInfo))
      // {
      //    id: 'peerId'
      //    multiaddrs: [
      //      '...'
      //    ]
      // }
